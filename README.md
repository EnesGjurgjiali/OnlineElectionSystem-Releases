## 🚀 About the Project
This desktop application serves as a secure kiosk interface for an online election system. It was built using:
* **Frontend:** React, Electron, Tailwind
* **Backend:** PHP, SQL

> **Note:** The source code for this project is currently held in a private repository for security/academic reasons. This repository serves exclusively for distributing the compiled desktop releases.

## ⚙️ Features
* Secure voter authentication
* Real-time vote processing
* Interactive and intuitive kiosk UI
